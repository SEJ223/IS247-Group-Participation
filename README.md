# IS247-Group-Participation
Separate repository for IS247 group task assignments.

Containerization for our project is hosted on Docker. You can find the Dockerfile in the /src folder. 

Docker deployment is automated using GitHub Actions with Docker. Due to the public nature of this repository, you are able to view the docker.yml file in the .github/workflows folder. 

You can view the JavaDocs at https://sej223.github.io/IS247-Group-Participation/
